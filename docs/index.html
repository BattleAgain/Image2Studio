Features

不只是生成按钮，而是一套图片创作流程。

从提示词、参考图、云任务、历史管理到保存分享，核心操作都在手机端闭环完成。

文生图

输入提示词即可调用 OpenAI 兼容图片接口生成作品，启动后自动获取模型并优先选择 gpt-image-2。

Cloud Mode

移动端长请求容易被系统网络/后台策略打断。Image2Studio 把耗时生成移动到 Cloudflare Worker 队列，手机只负责创建任务和取回结果。

Privacy

把敏感信息留在该留的地方。

Image2Studio 的设计原则很直接：不把真实密钥、签名证书、后台 token 放进仓库；用户 API Key 不写入 D1、KV 或日志；可选错误日志默认关闭，并清洗敏感字段。

不内置官方 API Key，避免公共额度被刷。

FAQ

常见问题

安装、接口、云任务、隐私相关问题先看这里。

需要自己准备 API Key 吗？

需要。Image2Studio 不内置公共 Key，避免别人刷额度。你可以使用 OpenAI 兼容图片接口的 Key。

为什么要做云任务模式？

图片生成耗时较长，手机后台和网络容易打断请求。云任务把长耗时处理放到 Worker Queue，App 只负责提交和取回结果。

云任务会保存我的 API Key 吗？

不会写入 D1、KV 或日志。Key 只在 Queue 消息中临时加密，用于本次任务消费。

能自己部署后端吗？

可以。仓库包含 Cloudflare Worker、D1 schema、迁移脚本和 wrangler 配置示例。

安装新版能覆盖旧版吗？

可以，前提是使用同一正式签名证书构建的 APK。官方 Release 会保持签名连续。

历史图片保存在哪里？

保存在 App 私有目录，支持保存到相册、分享、继续编辑和删除。
